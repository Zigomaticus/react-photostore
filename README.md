Простой интернет магазин на react`е. Можно добавлять/убирать товары в корзину и фаворты, делать заказы в корзине, смотреть все заказанные товары, использовать поиск.  
Используются хуки, роутер, fetch запросы на mockapi.
Под мобилку не адаптирован.
Поиграться можно тут https://zigomaticus.github.io/react-photostore/
